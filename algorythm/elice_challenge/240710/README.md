# 요구사항
주어진 수열의 부분 수열을 뽑아 그중 k번째로 큰 수 구하기

# 제한 사항
시간 제한 1초, 1<=n<=10000, 1<=m<=500 

# 접근법
n값이 크지 않아 인덱스 슬라이싱으로 접근해도 충분하다고 판단.



# 코드
```python
n, m = map(int, input().split())
arr = list(map(int, input().split()))
for _ in range(m):
    i , j , k = map(int, input().split())
    narr = arr[i-1:j]
    print(sorted(narr)[k-1])
    
```
